# CodeAlpha_Task2_Chatbot_messanger
This is Task 2 of my AI Internship at CodeAlpha. I have developed an intelligent FAQ Chatbot that uses Natural Language Processing (NLP) to understand user queries and provide the most relevant answers from a predefined dataset.
🚀 How it Works
Text Preprocessing: Uses NLTK for tokenization and removing stop words.

Similarity Matching: Utilizes Cosine Similarity (or simple keyword matching) to find the best answer.

Fallback System: Provides a "I'm sorry, I don't understand" response if no match is found.

🛠 Tech Stack
Python

NLTK / SpaCy (Natural Language Toolkit)

Scikit-learn (for TF-IDF Vectorization)
